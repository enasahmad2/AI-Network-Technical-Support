1. . مقدمة المشروع
نظام تفاعلي يعتمد على معمارية "عميل-سيرفر"، يهدف إلى أتمتة خدمات الدعم الفني التقني عبر دمج تقنيات الاتصال الشبكي (TCP Sockets) مع ذكاء اصطناعي توليدي (Google Gemini AI)، مما يوفر استجابة فورية وسياقية للمستخدمين.

2. الهيكلية التقنية (Technical Architecture)

- لغة البرمجة: Python 3.x

- بروتوكول الاتصال: TCP/IP (عبر مكتبة socket).

- نموذج الذكاء الاصطناعي: Google Gemini (1.5 Flash/Pro).

- المنفذ (Port): 5000.

- واجهة المستخدم: Tkinter (بتصميم Dark Mode عصري).

3. التحديات التقنية والحلول المبتكرة

أثناء التطوير، تم تجاوز عقبات تقنية حرجة عززت من استقرار النظام:
- تجاوز الحجب الجغرافي: تم استخدام VPN لتأمين قناة اتصال (Tunneling) مع خوادم جوجل، مما مكن النظام من العمل في بيئات مقيدة.

- التوافق البرمجي (Model Compatibility):
تم استبدال التعريفات الثابتة للموديلات بـ "استعلام ديناميكي"، حيث يقوم السيرفر عند الإقلاع بفحص الموديلات المتاحة في نسخة المكتبة المثبتة حالياً، مما منع خطأ 404 Not Found.

- استقرار تدفق البيانات: تم ضبط حجم الـ Buffer في عمليات الاستقبال والارسال لضمان عدم اقتطاع النصوص الطويلة الناتجة عن الذكاء الاصطناعي.

4. الميزات التقنية المتقدمة

- التعددية في المعالجة (Multithreading):
يعمل السيرفر بنظام Threading لتمكين عدة مستخدمين (لابتوبات مختلفة) من الاتصال والتحدث مع السيرفر في وقت واحد دون تعارض.

- إدارة الجلسات الذكية (Session Persistence):
يتم تخصيص جلسة محادثة (History) فريدة لكل مستخدم بناءً على الـ IP Address الخاص به، مما يسمح للنموذج بفهم سياق المحادثة الممتد.

- نظام التوثيق والرقابة (Logging System):
تم استبدال الـ Print التقليدي بنظام Logging متكامل يسجل كافة أحداث السيرفر (الاتصالات، الأخطاء، وحركة البيانات) في ملف server_activity.log لأغراض الصيانة والتدقيق.

5. دليل التشغيل عبر الشبكة (LAN Deployment)
لربط جهازين (السيرفر والعميل) عبر "الهوت سبوت":
1. ​جهاز السيرفر: تشغيل الـ VPN، ثم تنفيذ python server.py.
2. التعريف: استخراج الـ IP المحلي للجهاز (عن طريق أمر ipconfig).
3. جهاز العميل: تشغيل python gui-client.py وإدخال عنوان الـ IP الخاص بالسيرفر في واجهة البرنامج، ثم الضغط على "اتصال".

6. المتطلبات البرمجية
- اللغة: Python 3.10+

- المكتبات الخارجية: google-generativeai.

- المكتبات القياسية: logging, socket, threading, tkinter.
